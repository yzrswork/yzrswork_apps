や印工務店の道具箱

電子工作、自作PC、DIYと模型のための実用ツール。全てオフライン対応、インストール不要。

や印工務店の道具箱は、電子工作と自作PCを趣味にしているや印工務店(yzrswork)が、自分の作業で実際に必要になった「計算」「選び方」「切り分け」をそのまま使える形にした実用ツール集です。工具と部品ストックのチェックリスト、メモリやHDDの選び方ナビ、電源容量の見積もり、トラブル診断、プラモや模型の接着剤選び、外壁塗装やDIY塗装ができるタイミングの判定まで、すべて単一HTMLで動くPWAなので、一度開けばオフラインでも使えます。各ツールの背景には、部品の調達先えらび、はじめての自作PC、古い機材をよみがえらせる転生工作などを記録してきたnoteの記事があります。ツールで足りない詳しい経緯や失敗談は、noteのほうでどうぞ。

いちばん使われているツール
直し方ナビ

質問に答えるだけでトラブルを切り分ける診断ナビ。Yahoo!メール、Windows共有、自作PCビープ、Obsidian同期に対応。

電子工作

4 TOOLS
装備ナビ

電子工作の工具と部品ストックをステップ別にチェック。足りないものだけ買い物リストに集約。
工房の電卓

LED抵抗計算、カラーコード読み、555タイマー、電池駆動時間。作業台に常駐する計算ツール集。
はんだ付けナビ

はんだの種類と作業から適正なこて先温度を提示。イモはんだやブリッジのトラブル診断と道具選びつき。
ピンアサインナビ

Raspberry Pi、Arduino Uno、ESP32のピン配置を色分けで表示。3.3V注意やストラッピングピンの落とし穴つき。

自作PC

4 TOOLS
メモリ選びナビ

ソケットからDDR規格を確認し、用途から容量を選び、2枚構成で組むための選び方ナビ。
HDD選びナビ

WDのHDDを用途から色で選び、型番末尾でCMR/SMRを判定。NASにSMRを避ける早見つき。
自作PC構成プランナー

CPU、GPU、ストレージを選ぶと、電源容量の目安と構成サマリーが出る。組み立てチェックリストつき。
USB-C見分けナビ

手持ちのUSB-Cケーブルで何ができるか、用途に必要なのはどれか。W数、転送速度、映像出力を切り分け。

DIYと模型

3 TOOLS
接着剤選びナビ

くっつけたい素材を2つ選ぶだけで、接着剤タイプと推奨製品、注意点が分かる診断ナビ。素材の見分け方つき。
塗れるくん

住所を入れるだけで、外壁塗装やDIY塗装ができるタイミングを気温、湿度、露点、降水から判定。
ねじ下穴ナビ

タップの下穴、ボルトのバカ穴、木ねじの下穴。ねじのサイズを選ぶだけで開けるドリル径がわかる早見ナビ。
