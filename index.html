や印工務店のアプリたち

電子工作 x メイカー x AI。単一HTML製のニッチな道具を並べていきます。

PC / スマホ両対応

塗れるくん

住所を入れると、今日や数日先に外壁塗装ができるか(気温・湿度・露点・降水・風)を判定。

開く -
直し方ナビ

よくあるトラブルを質問に答えるだけで切り分ける診断ナビ。Yahoo!メール、Windows共有、自作PCビープ、Obsidian同期に対応。

開く -
装備ナビ

電子工作の工具と部品ストックをステップ別にチェックして、Amazonの買い物リストに集約。調達先ガイドつき。

開く -
Prompt Forge

キャラを崩さず、どの画像生成AIでもブレない完成プロンプトを一撃生成。

開く -
電子工作通信

HackadayとQiitaの電子工作記事をまとめて流す、自分用のRSSリーダー。

開く -
塗料量計算

面積・下地・塗料種別から必要リットル・塗り回数・概算を出す。塗れるくんと統合予定。

スマホ向け（タッチ操作）

YZRsynth

16ステップシーケンサー、5音色シンセ、ルーパーを1台にまとめたリズムマシン。

開く -
オタマトーン WEB

しっぽのリボンをなぞって鳴らす白いやつ。エフェクトとルーパー付き。

開く -
Grid Instrument

タップで鳴らす音楽グリッド。仕組みは単純、出てくる音は意外と複雑。

開く -
Metronome Watch

時計型ネオンのメトロノーム。背後を透明な振り子がゆっくり往復します。

開く -
Mizu

指でなぞって描く、水多めの水彩。にじんで混ざり、なぞるたびに水の音が落ちる。

開く -
花火

画面のどこを触っても、ちゃんと夏になる。幼児向けのタップゲーム。

開く -
あそぼう

モンテッソーリ風の知育ゲーム集。親はそっと隣で見ていてください。

開く -
ウィルスバスターズ

白血球の群れを操り、ウィルスの大軍を撃退する縦持ちディフェンス。
