# 塗れるくん

住所を入れると、今日や数日先に**外壁塗装ができるか**(気温・湿度・露点・降水・風)を判定するニッチツール。
単一HTML製のPWAで、GitHub Pages に置いて配布する。や印工務店(yzrswork)。

## いまの状態 (MVPの土台)

- `index.html` 単一ファイルで動作。外部ライブラリなし
- 住所検索: 国土地理院 ジオコーダ(キー不要・日本国内)
- 天気: Open-Meteo(キー不要・無料。hourly で気温/湿度/露点/降水/降水確率/風速)
- 判定: 作業時間帯(8-17時)の連続OK枠から「今日塗れる/注意/厳しい」を出し、おすすめ時間帯と数日先の見通しを表示
- PWA: `manifest.json` + `sw.js`(アプリ殻のみキャッシュ、APIは常にネット)

## 判定パラメータの調整

判定の閾値は `index.html` 冒頭の `CONFIG` ブロックに集約。各項目を **NG(施工不可)** と **WARN(中間=注意)** の2段で持つ。塗料種別で変わるので、現場感覚で最終確定する。

| 項目 | NG(施工不可) | WARN(注意) | 根拠 |
|---|---|---|---|
| 気温 | 5℃未満 / 35℃超 | 5-8℃ / 30-35℃ | JASS18・メーカー仕様: 推奨5-30℃、5℃未満は不可 |
| 湿度 | 85%以上 | 80-85% | JASS18・メーカー共通: 85%以上は不可 |
| 露点マージン(気温-露点) | 3℃未満 | 3-5℃ | 被塗面が露点+3℃を下回ると結露(ISO 8502-4 目安) |
| 降水量 | 0.1mm超 | - | その時間の降水 |
| 降水確率 | 60%超 | 30-60% | |
| 風速 | 10m/s超 | 7-10m/s | 飛散・乾燥ムラ |
| 作業時間帯 | - | - | 8-17時(WORK_START/END) |
| MIN_OK_HOURS | - | - | 連続OKが3時間で「今日OK」 |

> 閾値は JASS18塗装工事および各塗料メーカー仕様の一般的な目安で裏取りした初期値。塗料種別(水性/油性、上塗りの種類)で前後するため、や印さんの現場知識で調整すること。
